樂創預約綁定官方帳號說明-樂創預約系統

2024-06-21 樂創預約操作說明
樂創預約 樂創預約操作說明

1.進入 LINE官方帳號 後台

請使用電腦裝置進入 Line Official Account Manager

2.點選設定

登入官方帳號管理後台，會看到此畫面 ，點右上角 設定

3.進入 LINE Developers

點選左側選單 設定🔽 展開 > Messaging API > LINE Developers

點選右上角 Console，進入 LINE Developers 控制台

4.分享權限給樂創

點選 Provider下的帳號

進入設定頁面，點擊切換到 Roles

接著點擊 Invite by email

輸入邀請資訊
  • 樂創客服 mail：support@websiteszoo.com
  • Role：Admin

輸入完畢，Send invitations（此時右上角會出現 成功的通知訊息）

返回上層

點擊Channels下方框框，進入Messaging API 設定

點擊切換到最右側的 Roles設定

跟剛剛步驟一樣，點擊 Invite by email 輸入內容

  • 樂創客服 mail：support@websiteszoo.com
  • Role：Admin
    輸入完畢，Send invitations

5.等待樂創工程師設定

接著就是等待管理員設定囉！！過程約3-5分鐘

6.綁定完成

重新登入後，成功畫面如下
